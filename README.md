# AI Powered Farmer Query Analysis System

## Project Overview
This project uses Machine Learning and Natural Language Processing (NLP) to analyze farmer queries and generate agricultural insights.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Dataset
Raksha Farmer Query Dataset

## Features
- Query Analysis
- TF-IDF Vectorization
- Logistic Regression Classification
- Agricultural Insights
- Query Prediction System
